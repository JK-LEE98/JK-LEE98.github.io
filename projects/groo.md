# Groo

커뮤니티 기반 콘텐츠 서비스

---

# Tech Stack

- Spring Boot
- MyBatis
- Redis
- Vue.js

---

# Role

- MyBatis 기반 조회 기능 구현
- Frontend UI 개발
- Redis 기반 이미지 저장 구조 활용

---

# S3 활용 경험

이미지 데이터를 빠르게 조회하기 위해  
S3 기반 이미지 저장 구조를 활용했습니다.

Postman을 활용해 S3에 이미지 데이터를 업로드하고  
서비스에서 이미지 조회 기능을 테스트했습니다.
